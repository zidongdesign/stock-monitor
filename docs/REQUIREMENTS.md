## 一、股票自选股系统

### 自动补入
- 来源：热门板块个股（新浪 TOP10 板块）
- 只选：主板（60/00）+ 创业板（30）
- 排除：科创板（688）/ 北交所（8）/ ST / 近5日涨幅>30% / 财务D级
- 每天补入 5 只（按综合评分排序取 TOP 5）
- 可转债手动添加，不走自动扫描

### 自动淘汰
- 评分最低的 5 只自动移除
- 第 10 天起开始淘汰（前 9 天只补不淘）
- 被淘汰的股票如果重新符合条件，可再次被选入

### 自选池规则
- 初始：33 只 → 清洗为 5 只（评分 TOP 5）
- 上限：50 只
- 到上限后：先淘汰再补入

### 评分公式
- score = 财务面(40%) + 技术面(30%) + 资金面(30%)
- 财务评级：A/B/C/D
- score < 30 或 D 级 → 淘汰候选

### 每日收盘分析（15:35 自动）
- daily_analysis.py 对自选池打分
- sector_scan.py 扫描热门板块推荐个股
- 自动 git push → GitHub Pages 更新
